# form-demo

Experimenting with `react-beautiful-dnd` to create a From Builder

## How to run on localhost

First install dependencies:

```sh
yarn
```

To run in dev mode mode:

```sh
yarn start
```

Then go to http://localhost:8080

To create a production build:

```sh
yarn build
```
